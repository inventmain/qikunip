2025年10月29日，第26届中国专利奖评选通知发布。与往年相比，今年的变化是实质性的。

核心趋势是：专利本身的撰写质量是“入场券”，而申报单位的“全面合规性”被提升到前所未有的高度。

## 一、核心对比：第26届 VS 第25届

| **对比维度**        | **第26届**&lt;br&gt;**（今年）**                             | **第25届**&lt;br&gt;**（去年）** | **专业解读**                                                 |
| ------------------- | ------------------------------------------------------------ | -------------------------------- | ------------------------------------------------------------ |
| **1.政治/品德门槛** | 首次明确写入：“参评专利的专利权人和发明人（设计人）应坚决贯彻党的路线、方针和政策，模范遵守国家法律法规，社会信誉良好，具有良好的社会公德和职业道德。” | 无此项明确要求                   | 形式上的新增。这一要求将通过第3点的“多部门联合考察”机制落到实处，成为硬性约束。 |

|                |                                                              |                                                              |                                                              |
| -------------- | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **2.推荐渠道** | 大幅收紧。仅保留：&lt;br&gt;• 国务院各有关部门和单位&lt;br&gt;• 各省（自治区、直辖市）知识产权局&lt;br&gt;• 各有关全国性行业协会（不包括学会、商会）&lt;br&gt;• 中国科学院院士或中国工程院院士 | 渠道多样。包括：计划单列市、示范城市、示范园区、示范高校/企业（可自荐）等 | **“自荐”通道彻底关闭。** 省级局的审核权和推荐权变得至关重要；协会和院士的推荐权受到严格限制。 |

|                |                                                              |                                          |                                                              |
| -------------- | ------------------------------------------------------------ | ---------------------------------------- | ------------------------------------------------------------ |
| **3.评审流程** | 新增“考察”环节。流程：初审后提出建议名单，推荐单位需对建议名单中的全部专利权人和发明人（设计人）进行考察，征求纪检、公安等多部门意见。 | 流程相对简单：审核、公示后直接报送材料。 | **今年最核心的变化。** 合规审查空前严格：考察范围涵盖税务、环保、人社、市场监管、金融监管等多维度，任一环节有污点都可能一票否决。“非正常专利”严查：多部门数据联动下，历史问题难隐藏。 |

|                    |                                                              |                    |                                                              |
| ------------------ | ------------------------------------------------------------ | ------------------ | ------------------------------------------------------------ |
| **4.“非正常专利”** | 高压线。发现存在较大数量（比例）非正常专利申请的单位和个人，将取消其申报、推荐、参评或获奖资格。 | 条款内容基本一致。 | **执行环境发生重大变化。** 配合第3点的“多部门联合考察”机制，审查力度将显著加强，审查标准可能更严格。导向明确：优先推荐核心专利。 |

|                  |                                                              |                                                     |                                                              |
| ---------------- | ------------------------------------------------------------ | --------------------------------------------------- | ------------------------------------------------------------ |
| **5.优秀奖名额** | 固定数量。发明/实用新型优秀奖不超过700项；外观设计优秀奖不超过50项。 | 按比例。优秀奖获奖比例均不超过各类合格项目数的30%。 | **获奖总盘子固定，竞争规则更明确。** 如果合格项目数量过多，竞争将异常激烈。金奖、银奖名额保持不变。 |

|                  |                                                  |                                                   |                                                              |
| ---------------- | ------------------------------------------------ | ------------------------------------------------- | ------------------------------------------------------------ |
| **6.申报准备期** | 约3个月（2025年10月29日发布，2026年1月23日截止） | 约1个半月（2023年12月26日发布，2024年2月5日截止） | **时间显著延长，是去年的2倍。** 官方为“多部门联合考察”和企业自查合规性问题预留了充足时间。 |

|              |                                              |                      |                                                              |
| ------------ | -------------------------------------------- | -------------------- | ------------------------------------------------------------ |
| **7.申报书** | 使用“2025年修订版”，仅经济效益统计年份更新。 | 使用“2023年修订版”。 | **核心内容几乎没有变化。** 评价板块（专利质量、技术先进性、运用成效等）和字数限制完全一致。申报材料的撰写质量依然是获奖的基础。 |

## 二、应对策略

### 1\. 立即启动合规自查（最重要）

根据第3点的考察范围，企业需首先自查：

- 税务合规：无欠税、无重大税务违法记录（可通过电子税务局查询纳税信用等级）。
- 环保合规：无环保行政处罚、无环保信访投诉（可查询各地环保部门公开的处罚信息）。
- 安全生产：无安全生产事故、无重大安全隐患（可查询应急管理部门的公开信息）。
- 劳动用工：社保缴纳合规、无重大劳动纠纷（可查询人社部门信用记录）。
- 市场监管：无重大行政处罚、无严重失信行为（可通过“国家企业信用信息公示系统”查询）。
- 金融监管：无金融违法违规行为。

**建议做法：** 建立自查清单，逐项核对，发现问题及时整改并保留整改证据。

### 2\. 排查“非正常专利”历史

严格对照《关于规范申请专利行为的办法》，排查：

- 是否存在非正常专利申请行为。
- 是否存在专利权权属纠纷、侵权纠纷。
- 是否存在专利权被宣告无效的情况。

### 3\. 核查人员资格

确保所有专利权人和发明人：

- 无违法犯罪记录。
- 无严重失信行为。
- 无重大行政处罚记录。

### 4\. 精细打磨申报材料

申报材料的撰写质量依然是获奖的基础：

- 尽早准备经济效益专项审计报告。
- 完善专利许可合同备案证明、质押登记通知书等佐证材料。
- 要求放宽：专利密集型产品备案证明已从“应当提交”改为“可以提交”。
- 重点突出专利的产业化效益，尤其是在基础研究、前沿科技、“卡脖子”技术等方面的贡献。

## 三、建议时间表

- **11月：** 启动内部自查，排查合规性风险和非正常专利问题；完成联系人报名（11月7日前）。
- **12月：** 准备和撰写申报材料，完善经济效益数据和证明文件。
- **1月上旬：** 完成材料报送（截止日期：2026年1月23日）。

## ⚠️ 其他重要提醒

- **协会推荐限制：** 每个协会仅限推荐3项，且协会“最近一次全国性年度检查结论应为合格”；“连续两届推荐项目未获奖的协会，暂停推荐资格一年”。
- **院士推荐限制：**“同专业领域的两名院士可联名推荐1项本专业领域的发明专利，每名院士仅限推荐1次”。
- **推荐单位的责任：** 通知明确要求推荐单位确保相关材料完整、真实、准确，且集体研究审慎提出推荐项目，这意味着推荐单位对推荐项目的审核会更加严格。
- **材料报送方式：**“材料统一采用EMS快递方式报送，不接受现场申报，材料不符合要求或逾期（以快递寄出日为准）推荐的项目均不予受理”。
- **官方文件下载：** 国家知识产权局政府门户网站“中国专利奖”专栏（https://www.cnipa.gov.cn/col/col41/index.html）。
